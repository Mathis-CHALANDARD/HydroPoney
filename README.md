
# Hydroponey 🌊

Ce projet a pour but de rendre connectée une serre hydroponique.\
L'utilisateur pourra voir des données de la serre en temps réel, et agir avec l'éclairage de cette dernière.




## Technologies utilisées ⚙
📱 Téléphone Android \
📻 Arduino MRK1310 \
📻 Raspberry Pi\
📡 MQTT
